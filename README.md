`positional_polarization.pdf` - текст анкеты кросс-секционного опроса

`S2_data_raw.xlsx` - массив данных с результатами кросс-секционного опроса

**`cross_sec.ipynb`** - код для анализа результатов кросс-секционного опроса

**`vignette_generation.ipynb`** - код для генерирования виньеток

`виньетки_итог.xlsx` - наборы виньеток, использовавшиеся при составлении онлайн-эксперимента

`polarization_experiment.pdf` - текст анкеты для онлайн-эксперимента

`anketa523184-2026-05-26.xlsx` - выгруженные с 1ka.si сырые данные, собранные в ходе онлайн-эксперимента

**`preprocessing_for_experiment_analysis.ipynb`** - код для подготовки массивов данных для анализа результатов онлайн-эксперимента

`long_ИТОГ.xlsx` - массив данных, где одно наблюдение - это оценка респондентом одного кандидата (по 14 строк на респондента)

`pair_ИТОГ.xlsx` - массив данных, где одно наблюдение - это оценка респонеднтом пары кандидатов из одной виньетки (по 7 строк на респондента)

**`experiment_analysis.ipynb`** - код для анализа результатов онлайн-эксперимента
